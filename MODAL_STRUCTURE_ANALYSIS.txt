수신자 추가 모달 3개 구조 분석
================================================================================

📷 첫 번째 이미지 - 주소록 모달 (빈 상태)
================================================================================

[헤더 영역]
- 좌측: "수신자 목록에 연락처 추가" (제목, 검은색)
- 우측: X 닫기 버튼 (회색)

[좌측 상단 탭/버튼 영역]
- "새 그룹" 버튼 (회색 배경)
- "엑셀 업로드" 버튼 (회색 배경)
- "그룹 삭제" 버튼 (회색 배경, 비활성화 상태로 보임)
- 버튼 그룹 하단: 검색창 + "그룹명 검색" placeholder + 돋보기 아이콘

[좌측 메인 컨텐츠 영역] (빈 상태, 회색 배경)
- 중앙 정렬
- "조회된 연락처 그룹이 없습니다." (큰 텍스트)
- 아이콘 없음

[좌측 하단 정보 텍스트]
- "그룹 0개 선택됨"
- 그룹 이름을 우클릭하여 그룹을 관리할 수 있습니다 (텍스트)

[우측 상단 영역]
- 추가한 수신번호(총 0개)
- "비우기" 버튼 (회색 배경, 비활성화 상태로 보임)

[우측 메인 컨텐츠 영역] (빈 상태)
- 중앙 정렬
- "수신자명단이 비어있습니다." (큰 텍스트)
- 아이콘 없음

[하단 푸터 영역]
- 우측 하단:
  - "수신자 명단에 추가" 버튼 (회색 또는 비활성 상태로 보임)
  - "채팅 문의" 버튼
  - "닫기" 버튼


================================================================================
📷 두 번째 이미지 - 엑셀 업로드 모달
================================================================================

[헤더 영역]
- 좌측: "수신자 목록에 연락처 추가" (제목, 검은색)
- 우측: X 닫기 버튼 (회색)

[메인 컨텐츠 영역]

  [상단 섹션]
  - 상단:
    - 초록색 엑셀 아이콘 (Excel 아이콘)
    - "엑셀 파일 업로드 (csv, xls, xlsx)" (검은색 텍스트)
  - 하단:
    - "예제파일 내려받기" 버튼 (흰 배경, 검은 테두리, 다운로드 아이콘 포함)

  [드래그 앤 드롭 영역] (밝은 회색 배경)
  - 업로드 아이콘 (회색, 중앙 상단)
  - "엑셀 파일 업로드" (큰 검은 텍스트, 중앙)
  - "이곳에 파일 끌어오기 혹은 찾아보기" (작은 회색 텍스트, 중앙)

  [하단 옵션 영역]
  - 상단
     - 토글 스위치 (OFF 상태, 회색)
     - 전화번호부 아이콘
     - "주소록 생성 후 저장하기" (검은색 텍스트)
     - 물음표 도움말 아이콘
  - 하단
     - 좌측: "수신자 목록에 추가" 버튼 (보라색 배경, 흰 글씨)
     - 우측: "미리보기" 버튼 (흰 배경, 검은 테두리)

[하단 푸터 영역]
  - "채팅 문의" 버튼 (달 아이콘)
  - "닫기" 버튼
