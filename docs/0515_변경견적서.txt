# 📌 B2C/B2B 메시지 발송 웹포탈 구축 프로젝트 변경 견적서

## ✅ 프로젝트 개요
* **프로젝트명**: 메시지 발송 웹포탈 개발 (확장)
* **목표**: 캠페인 중심 메시지 발송 UI 및 관리 시스템 구축 (PG, 정산, 리워드 포함) + AI 기능 강화 + 추가 메시징 채널 지원
* **개발 기간**: 
  * 1차 개발: 총 10주 (기존 기능 + AI 기능 강화)
  * 2차 개발: 추가 4주 (RCS, 알림톡, 친구톡플러스 확장)
* **개발 비용**: 
  * 1차 개발: 25,000,000원 (부가세 별도)
  * 2차 개발: 10,000,000원 (부가세 별도)
  * 총 비용: 35,000,000원 (부가세 별도)

## 🗂️ 주요 개발 범위

### 1. 기획 및 설계
* PC/MO 반응형 웹 흐름 정의
* 화면 플로우 및 와이어프레임 설계
* ERD 및 API 스펙 기획
* 요구사항 명세서 및 구현 범위 세부 정의
* **추가**: AI 기능 관련 데이터 스키마 및 흐름 설계
* **추가**: 확장 메시징 채널(RCS, 알림톡, 친구톡플러스) 연동 아키텍처 설계

### 2. UI/UX 디자인
* 관리자 / 사용자 / 캠페인 관리 페이지 디자인
* 반응형 웹 기준 (PC/MO 대응)
* 레퍼런스 사이트 기반 UI/UX 최적화
* **추가**: 템플릿 그래픽 에디터 UI/UX 디자인 (글자 폰트, 크기, 위치 변경 기능)
* **추가**: AI 템플릿 생성 및 캠페인 자동화 UI/UX 디자인

### 3. 프론트엔드
* Next.js + TypeScript 기반 웹 애플리케이션 개발
* 캠페인 등록, 메시지 내용 작성, 대상자 업로드
* 회원가입 및 인증, 사용자 대시보드 구현
* 관리자 전용 기능 및 인터페이스 구현
* 반응형 디자인 적용 (모바일/데스크톱)
* **추가**: 그래픽 에디터 구현 (템플릿 제작 시 글자 폰트, 크기, 위치 변경 기능)
* **추가**: AI 템플릿 추천 및 자동완성 UI 구현
* **추가**: 채널 간 템플릿 자동 연동 기능 구현 (한 채널에서 작성된 템플릿이 다른 채널에 자동 적용)

### 4. 백엔드
* Spring Boot + PostgreSQL + Java 기반 API 서버 개발
* 사용자 / 캠페인 / 결제 / 리워드 / 정산 로직 구현
* 메시지 발송 요청은 인터페이스 테이블(DB) 적재만 진행
* 관리자 API 개발 (승인, 검토, 통계 집계)
* RESTful API 설계 및 개발
* **추가**: AI 모델 연동 및 API 개발 (GPT 등 외부 AI 서비스 연동)
* **추가**: 템플릿 자동 생성 및 추천 알고리즘 개발
* **추가**: 캠페인 타겟팅 자동화 및 추천 알고리즘 개발
* **추가**: 사용자 행동 데이터 로깅 및 분석 시스템 (AI 고도화 기반)

### 5. 관리자 시스템
* 별도 관리자 웹 콘솔 개발
* 사용자 관리 및 권한 제어
* 캠페인 승인/반려 프로세스
* 발신번호 검수 및 승인 기능
* 기업서류 검수 및 승인 기능
* 통계 및 대시보드 제공
* **추가**: AI 생성 템플릿 및 캠페인 검수 기능

### 6. PG 및 정산 연동
* 카드 결제 연동 (PG사 연동)

* 세금계산서 발행용 자료 생성 기능

### 7. 영업사원 리워드 시스템 (다단계)

* 다단계 MLM(Multi-Level Marketing) 구조 구현
* 직접 추천 및 간접 추천에 따른 차등 수수료 구조 설계
* 계층별(레벨별) 수수료 비율 설정 기능

* 영업사원 대시보드 (직접/간접 추천 현황, 수익 현황, 조직도)

### 8. AI 기능 강화 (추가)
* 회원정보 기반 템플릿 자동 추천 시스템
* 회원정보 기반 캠페인 타겟팅 자동 추천 시스템

## 📎 특이사항 및 유의점
* 메시지 서버는 의뢰인 측에서 제공하며, 메시지 발송은 인터페이스 테이블(DB) 적재만 진행
* 외부 시스템 연동은 클라이언트 제공 스펙 기준으로 대응
* 디자인 리소스(로고, 브랜드 컬러 등) 클라이언트 제공 전제
* PG사는 클라이언트 사전 계약 완료되어야 연동 가능
* 레퍼런스 사이트(solapi.com, ppurio.com, findjob.co.kr) 참고하여 UI/UX 구성
* 영업사원 리워드 시스템은 다단계(MLM) 구조로 구현하여 직접 및 간접 추천에 따른 차등 리워드 제공
* **추가**: RCS, 알림톡, 친구톡플러스는 8월 오픈 이후 2차 개발 범위에 포함 (14주 내 전체 완료)
* **추가**: AI 기능은 1차 개발(10주) 내에 완료하여 8월 오픈 시점에 포함
* **추가**: AI 서비스 연동 비용은 별도로 발생할 수 있으며, 클라이언트 계정으로 연동 가정
