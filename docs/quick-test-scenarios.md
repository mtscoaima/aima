# MTS 메시징 플랫폼 - 빠른 테스트 시나리오

## 🚀 30분 핵심 기능 테스트

이 문서는 외주 클라이언트가 30분 내에 핵심 기능을 빠르게 테스트할 수 있도록 구성되었습니다.

---

## 시작하기 전에
1. 터미널에서 개발 서버 실행:
```bash
cd /Users/johnlee12/Desktop/git/mts-message
npm run dev
```
2. Chrome 브라우저에서 http://localhost:3000 열기
3. 팝업 차단 해제 설정

---

## 시나리오 1: 신규 사용자 여정 (10분)

### 1단계: 회원가입
1. **http://localhost:3000/signup** 접속
2. 일반회원 선택
3. 다음 정보로 가입:
   - 아이디: newuser001
   - 비밀번호: Test@1234
   - 이름: 김테스트
   - 이메일: test@example.com
   - 휴대폰: 010-1234-5678
   - 회사명: 테스트회사
   - 사업자번호: 123-45-67890
4. 전체 동의 체크
5. 회원가입 버튼 클릭

### 2단계: 첫 로그인
1. 방금 만든 계정으로 로그인
2. 대시보드 확인
3. 사업자 인증 배너 확인

### 3단계: 기본 설정
1. 상단 프로필 → 마이페이지 클릭
2. 발신번호 관리 탭 → 발신번호 추가
3. 02-1234-5678 입력 후 저장

**✅ 확인 포인트**: 회원가입 → 로그인 → 기본 설정 완료

---

## 시나리오 2: AI 마케팅 활용 (10분)

### 1단계: AI 상담
1. 상단 메뉴 "AI 타겟마케팅" 클릭
2. 채팅창에 입력: "카페 신메뉴 홍보 문자 만들어줘"
3. AI 응답 대기 (5-10초)

### 2단계: 템플릿 수정
1. "더 짧게 만들어줘" 추가 요청
2. AI가 수정한 내용 확인

### 3단계: 템플릿 저장
1. "템플릿 저장" 버튼 클릭
2. 템플릿명: "카페 신메뉴 홍보"
3. 저장 완료

### 4단계: 캠페인 확인
1. "캠페인 관리" 탭 클릭
2. 방금 저장한 캠페인 확인

**✅ 확인 포인트**: AI 대화 → 템플릿 생성 → 저장 완료

---

## 시나리오 3: 고객지원 이용 (5분)

### 1단계: FAQ 검색
1. 상단 메뉴 "고객센터" 클릭
2. "자주 묻는 질문" 탭 클릭
3. 검색창에 "결제" 입력
4. 관련 FAQ 확인

### 2단계: 문의하기
1. "문의하기" 탭 클릭
2. 간단한 문의 작성:
   - 유형: 서비스 이용
   - 제목: 테스트 문의
   - 내용: 테스트입니다
3. SMS 알림 체크
4. 문의하기 버튼 클릭

**✅ 확인 포인트**: FAQ 검색 → 문의 등록 완료

---

## 시나리오 4: 관리자 기능 (5분) - 관리자 계정 필요

### 준비: 관리자 로그인
1. 로그아웃 후 관리자 계정으로 로그인
   - 아이디: admin001
   - 비밀번호: Admin@1234

### 1단계: 회원 관리
1. 좌측 사이드바 "회원관리" 클릭
2. 검색창에 "test" 입력
3. 회원 목록 확인
4. 첫 번째 회원 "상세보기" 클릭

### 2단계: 일괄 처리
1. 2-3명 체크박스 선택
2. "일괄 처리" 버튼 클릭
3. 등급 변경 선택
4. 적용

**✅ 확인 포인트**: 회원 검색 → 상세 정보 → 일괄 처리

---

## 🎯 5분 스피드 테스트

가장 핵심적인 기능만 빠르게 확인:

1. **로그인** (1분)
   - testuser001 / Test@1234

2. **AI 채팅** (2분)
   - AI 타겟마케팅 → "이벤트 문자 만들어줘" → 응답 확인

3. **마이페이지** (1분)
   - 프로필 → 마이페이지 → 각 탭 클릭해서 확인

4. **고객센터** (1분)
   - 고객센터 → 공지사항 1개 클릭 → FAQ 검색 "결제"

## ⚡ 성능 체크포인트

각 동작의 예상 소요 시간:
- 페이지 로딩: 1-2초
- AI 응답: 5-10초
- 데이터 저장: 1초 이내
- 검색 결과: 즉시

⏰ 이보다 오래 걸리면 이슈로 기록

---

## 🐛 자주 발생하는 이슈와 해결법

| 증상 | 해결 방법 |
|------|----------|
| 로그인 안 됨 | 아이디/비밀번호 확인, Caps Lock 확인 |
| AI 응답 없음 | 10초 대기 후 페이지 새로고침 |
| 버튼 클릭 안 됨 | 페이지 완전 로딩 대기 |
| 팝업 안 열림 | 브라우저 팝업 차단 해제 |
