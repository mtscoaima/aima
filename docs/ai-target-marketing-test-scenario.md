# AI 타겟 마케팅 테스트 시나리오

## 개요
AI 타겟 마케팅 기능의 새로운 플로우를 테스트하기 위한 시나리오입니다.

## 테스트 시나리오

### 1. 첫 대화 - 질문 생성 확인
1. `/target-marketing` 페이지 접속
2. AI 대화 시작
3. 첫 메시지 입력 예시: "카페 홍보를 하고 싶어요"
4. **확인사항:**
   - AI가 3개의 질문을 생성하는지 확인
   - 캠페인 목적에 대한 질문이 포함되었는지 확인
   - 혜택/프로모션에 대한 질문이 포함되었는지 확인
   - 이미지가 생성되지 않는지 확인
   - 질문에 선택지가 없는지 확인

### 2. 질문에 대한 응답 - 콘텐츠 생성 확인
1. AI의 질문에 대한 답변 입력
   예시: "신규 고객 유치가 목적이고, 아메리카노 1+1 이벤트를 진행하려고 합니다. 20-30대 직장인들이 주 타겟입니다."
2. **확인사항:**
   - 마케팅 이미지가 생성되는지 확인
   - SMS 텍스트가 생성되는지 확인
   - SMS 텍스트가 90자 이내, 2줄로 제한되는지 확인
   - 혜택(1+1)과 목적(신규 고객)이 앞부분에 배치되는지 확인
   - 퀵 액션 버튼이 생성되는지 확인

### 3. 추가 대화 - 일반 콘텐츠 생성
1. 추가 요청 입력
   예시: "디저트 할인 이벤트도 추가로 만들어주세요"
2. **확인사항:**
   - 질문 없이 바로 콘텐츠가 생성되는지 확인
   - 이전 대화 맥락을 유지하는지 확인

## 예상 결과

### 첫 대화 응답 예시:
```
안녕하세요! 효과적인 마케팅 캠페인을 만들어드리기 위해 몇 가지 질문을 드리겠습니다.

1. 이번 카페 홍보의 주요 목적은 무엇인가요? (예: 신규 고객 유치, 기존 고객 재방문 유도, 특정 메뉴 홍보 등)

2. 고객님께 제공하실 특별한 혜택이나 프로모션은 무엇인가요? (예: 할인율, 무료 제공 품목, 이벤트 기간 등)

3. 주로 타겟하시는 고객층은 어떤 분들인가요? 그리고 카페의 특별한 강점이나 차별점이 있다면 말씀해주세요.
```

### 두 번째 응답 예시:
- **생성된 이미지**: 커피 1+1 이벤트 관련 이미지
- **SMS 텍스트**: 
  ```
  [1+1 이벤트] 신규고객 아메리카노 1+1!
  20-30대 직장인 환영, 오늘부터 한정 진행
  ```
- **퀵 액션 버튼**: "이벤트 확인하기", "매장 위치", "메뉴 보기", "예약하기"

## 테스트 체크리스트
- [ ] 첫 대화에서 3개의 질문이 생성됨
- [ ] 질문에 선택지가 없음
- [ ] 첫 대화에서 이미지가 생성되지 않음
- [ ] 두 번째 대화부터 이미지와 콘텐츠가 생성됨
- [ ] SMS 텍스트가 2줄 이내로 제한됨
- [ ] SMS 텍스트가 90자 이내로 제한됨
- [ ] 혜택과 목적이 텍스트 앞부분에 배치됨
- [ ] 퀵 액션 버튼이 적절히 생성됨
