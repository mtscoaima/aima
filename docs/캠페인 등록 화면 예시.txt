🖼️ 사용자 화면 예시 – 01: 템플릿 생성

1. 기본 정보 입력
	•	제목 입력
Input field – 사용자가 자유롭게 제목을 입력
	•	문구 입력
Textarea or Input field – 광고 문구 직접 입력
	•	문구 선택
	•	🔘 내 문자함에서 선택 (라디오 버튼)
	•	🔘 추천 문구 사용 (라디오 버튼)

2. 부가 옵션
	•	특수문자 추가
Dropdown or Icon button – 선택 시 특수문자 삽입
	•	변수 추가
Dropdown – {이름}, {날짜} 등 삽입 변수 제공

3. 이미지 등록
	•	파일 업로드 버튼 – PNG, JPG 등 이미지 업로드 가능

4. 발신번호 선택
	•	Dropdown 또는 Input field – 발신번호 선택 또는 직접 입력

5. 수신 대상 입력
	•	주소록 선택
	•	엑셀 업로드
	•	메모장 입력

6. 발송 옵션
	•	🔘 즉시 발송
	•	🔘 예약 발송
	•	🔘 분할 발송

7. 기타
	•	✅ 중복 제거 (체크박스)
	•	🔍 미리보기 버튼
	•	🚀 발송하기 버튼
	•	🧪 테스트 전송 버튼

🖼️ 사용자 화면 예시 – 02: AI 템플릿 및 타겟 생성

1. 캠페인 정보 입력
	•	캠페인 제목 입력
Input field
	•	캠페인 기간 선택
Date picker (시작일 ~ 종료일)

2. AI 기반 추천 옵션
	•	성별 선택
	•	⬜ 남성
	•	⬜ 여성
	•	⬜ 전체
	•	연령 선택
	•	체크박스 형태 (10대, 20대, 30대, 40대, 50대 이상)
	•	지역 선택
	•	다중 선택 Dropdown 또는 Checkbox
	•	업종 선택
	•	다중 선택 Dropdown 또는 Checkbox

3. AI 추천 문구/이미지 영역
	•	자동 생성된 이미지/문구 예시 표시 영역 (미리보기 제공)

4. 버튼
	•	🔄 다시 생성하기 (AI 재생성 요청)
	•	✅ 이 템플릿 사용하기
