# 1. 개요

본, API 문서는 카카오 비즈 메시지(알림톡/친구톡)를 발송하기 위한 카카오톡 채널 발신프로필키를
고객사 시스템에서 관리하기 위하여 제공하여 드리는 API 문서입니다.

본, API 를 이용하실 경우 카카오 채널 발신프로필키에 대한 관리 책임은 고객사에 있으며, 이로
인하여 발생하는 불이익에 대하여 (주)엠티에스컴퍼니는 책임을 지지 않습니다. 또한 발신프로필키
도용, 임의 삭제 등의 문제가 발생하였을 경우 민·형사상에 책임이 있음을 안내 드립니다.

본, API 매뉴얼은 허가된 업체만 사용이 가능하며, 무단 복제, 전제, 배포를 금합니다.

# 2. 준비사항

본, API 를 사용하기 위해서는 고객사 인증을 위하여 "authCode"(고객사 인증코드)를 영업담당자
에게 신청하여 발급 받아야 합니다.

authCode(고객사 인증코드) : 고객사 인증을 위한 고유 코드

# 3.API 기본 사항

- ○ 운영 URL : https://talks.mtsco.co.kr
- 전송 메소드 : POST
- 전송 방식 : multipart/form-data


# 4. 발신프로필 관리

# 4.1 카카오톡 채널 인증 토큰 요청

- ○ 발신프로필 등록을 위한 카카오톡 채널 인증 토큰을 요청하는 API 입니다.


- ○ 입력한 전화번호에 연결된 카카오톡으로 인증 토큰이 발송되며, 발송된 토큰은
- 발신프로필 등록 시 사용됩니다.


- ○ 인증받은 토큰은 7 일동안 비즈메시지 센터 서버에 보관됩니다.


# [REQUEST] httpss://talks.mtsco.co.kr/mts/api/sender/token

[Parameter]

| 키 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| authCode | text | Y | 고객사 인증코드 |
| yellowld | text | Y | 카카오톡 채널 (예: @에릭커피) |
| phoneNumber | text | Y | 카카오톡 채널 알림받는 관리자 핸드폰 번호 |


# [RESPONSE]

| 키 | 타입 | 설명 |
| --- | --- | --- |
| code | text | 결과 코드 |

# 5. 등록

- ○ 발신프로필을 신규 등록합니다.


- ○ 사전에 카카오톡 채널이 생성되어 있어야 하며, 카카오톡 채널 관리자센터에서 비즈니스 인증을
- 받아야 합니다.


- ○ 카카오톡 채널은 프로필 activated 상태이고 운영자에 의해 차단되지 않은 정상 상태여야 등록
- 가능합니다.


- ○ 카카오톡 채널로 관리자 인증 토큰(2.1. 카카오톡 채널 인증 토큰 요청)을 등록 전 받아야 하며,
- 토큰을 요청한 전화번호로만 발신프로필로 등록 가능합니다.


- ○ 2.2. 카테고리 전체 목록 조회 API 를 사용하여 입력할 카테고리 코드 목록을 조회할 수 있으며,
- 카테고리 정보를 함께 등록할 수 있습니다.
