# □ 파일 업로드 API

- - 운영 URL: https://talks.mtsco.co.kr


- - 전송 메소드: POST


- - 전송 방식 : multipart/form-data


1. 알림톡 템플릿 등록용 이미지 업로드 요청

- · 강조유형이 이미지형 또는 아이템리스트형인 알림톡 템플릿 등록 시 사용될 이미지를 업로드
- 합니다.
- · 제한 사이즈 - 가로 500px 이상, 가로:세로 비율 2:1
- · 파일형식 및 크기 : jpg, png / 최대 500KB


# [Request] https://talks.mtsco.co.kr/mts/api/image/alimtalk/template

# [Parameter]

| 키 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| senderKey | text | Y | 발신 프로필 키 (senderKeyType 이 G 인경우발신 프로필그룹키) |
| senderKeyType | text | N | 발신 프로필 키 타입 (G:그룹 , S:발신프로필(default)) |
| image | binary | Y | 업로드할 이미지 파일 |


# [Response]

응답 바디는 JSON 객체로 아래 값을 참고 해주세요. 요청 성공이 아닐 경우 message 로 설명이
포함됩니다.

| 키 | 타입 | 설명 |
| --- | --- | --- |
| code | text | 결과 코드 |
| image | text | 업로드된 이미지 URL |


# 2. 알림톡 하이라이트 이미지 업로드 요청

- · 아이템리스트 알림톡 발송 시 사용될 아이템 하이라이트 이미지를 업로드 합니다.
- · 제한 사이즈 - 가로 108px 이상, 가로:세로 비율이 1:1
- · 파일형식 및 크기 : jpg, png / 최대 500KB


[Request] https://talks.mtsco.co.kr/mts/api /image/alimtalk/itemHighlight

# [Parameter]

| 키 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| senderKey | text | Y | 발신 프로필 키 |
| senderKeyType | text | N | 발신프로필타입 (G:그룹, S:기본(default)) |
| image | binary | Y | 업로드할 이미지 파일 |


# [Response]

응답 바디는 JSON 객체로 아래 값을 참고 해주세요. 요청 성공이 아닐 경우 message 로 설명이
포함됩니다.

| 키 | 타입 | 설명 |
| --- | --- | --- |
| code | text | 결과 코드 |
| data | text | 발신프로필키 |


응답코드

| code | 설명 |
| --- | --- |
| 0000 | 요청 성공 |
| 1004 | 필수 파라미터가 없음 |
| 2001 | 이미지 용량 초과 |
| 2002 | 발송할 수 없는 이미지 사이즈 |
| 2003 | 지원하지 않는 이미지 형식 |
| 2004 | 카카오 내부 시스템 오류로 업로드 실패 |
| 999 | 엠티에스 - 카카오톡 연동 오류 및 기타 오류 |
