# MTS 메시징 플랫폼 - 수동 테스트 가이드

## 📋 테스트 준비사항

### 1. 개발 서버 실행
1. 터미널을 열고 프로젝트 폴더로 이동합니다
2. 다음 명령어를 실행합니다:
```bash
cd /Users/johnlee12/Desktop/git/mts-message
npm install
npm run dev
```
3. 브라우저에서 http://localhost:3000 접속

### 2. 테스트 환경 설정
- **권장 브라우저**: Chrome 또는 Edge
- **화면 해상도**: 1920x1080 이상
- **팝업 차단 해제**: 브라우저 주소창 우측 팝업 차단 아이콘 클릭 → "항상 허용"

### 3. 테스트 계정 정보
```
일반 사용자:
아이디: testuser001
비밀번호: Test@1234

영업사원:
아이디: testsales001
비밀번호: Sales@1234

관리자:
아이디: admin001
비밀번호: Admin@1234
```

## 🧪 기능별 수동 테스트 절차

## 1. 회원가입 테스트

### 1.1 일반회원 가입
1. **회원가입 페이지 접속**
   - 브라우저에서 http://localhost:3000/signup 접속
   - 또는 로그인 페이지에서 "회원가입" 링크 클릭

2. **회원 유형 선택**
   - "일반회원" 라디오 버튼 선택
   - 추천인 코드가 있다면 입력 (선택사항)

3. **기본 정보 입력**
   - 아이디: 영문, 숫자 조합 (예: testuser002)
   - 비밀번호: 대소문자, 숫자, 특수문자 포함 8자 이상 (예: Test@1234)
   - 비밀번호 확인: 위와 동일하게 입력
   - 이름: 실명 입력
   - 이메일: 유효한 이메일 형식
   - 휴대폰번호: 010-0000-0000 형식

4. **회사 정보 입력**
   - 회사명: 테스트 회사
   - 사업자등록번호: 123-45-67890

5. **약관 동의**
   - "전체 동의" 체크박스 클릭
   - 또는 개별 약관 확인 후 각각 동의

6. **회원가입 완료**
   - "회원가입" 버튼 클릭
   - 성공 메시지 확인
   - 로그인 페이지로 자동 이동 확인

### 1.2 영업사원 가입
1. 회원가입 페이지에서 "영업사원" 라디오 버튼 선택
2. 나머지 절차는 일반회원과 동일
3. 가입 후 관리자 승인 대기 상태 확인

## 2. 로그인 테스트

### 2.1 일반 로그인
1. **로그인 페이지 접속**
   - http://localhost:3000/login 접속

2. **로그인 정보 입력**
   - 아이디: testuser001
   - 비밀번호: Test@1234

3. **로그인 시도**
   - "로그인" 버튼 클릭
   - 대시보드로 이동 확인

### 2.2 소셜 로그인
1. **카카오 로그인**
   - 노란색 "카카오 로그인" 버튼 클릭
   - 카카오 계정으로 로그인
   - 첫 로그인 시 추가 정보 입력

2. **네이버 로그인**
   - 초록색 "네이버 로그인" 버튼 클릭
   - 네이버 계정으로 로그인

3. **구글 로그인**
   - "구글 로그인" 버튼 클릭
   - 구글 계정으로 로그인

### 2.3 로그인 상태 확인
- 상단 헤더에 사용자 이름 표시 확인
- 알림 아이콘 표시 확인
- "로그아웃" 버튼 표시 확인

---

## 3. AI 타겟마케팅 테스트

### 3.1 AI 채팅 상담
1. **AI 타겟마케팅 페이지 접속**
   - 상단 메뉴에서 "AI 타겟마케팅" 클릭
   - 또는 http://localhost:3000/target-marketing 직접 접속

2. **AI와 대화하기**
   - 하단 입력창에 "미용실 개업 홍보 문자를 만들어주세요" 입력
   - Enter 키 또는 전송 버튼 클릭
   - AI 응답 확인 (약 5-10초 소요)

3. **추가 요청하기**
   - "좀 더 친근하게 수정해주세요" 입력
   - AI가 수정된 내용 제공하는지 확인

### 3.2 이미지 업로드
1. **이미지 첨부 버튼 클릭**
   - 입력창 좌측 클립 아이콘 클릭
   - 이미지 파일 선택 (JPG, PNG)
   - 업로드 완료 확인

### 3.3 템플릿 저장
1. **템플릿 저장 버튼 클릭**
   - AI가 생성한 템플릿 하단의 "템플릿 저장" 버튼 클릭
   
2. **저장 정보 입력**
   - 템플릿명: "미용실 개업 홍보"
   - 카테고리 선택
   - "저장" 버튼 클릭

### 3.4 캠페인 관리
1. **캠페인 관리 탭 클릭**
   - 페이지 상단 "캠페인 관리" 탭 클릭

2. **저장된 캠페인 확인**
   - 목록에서 방금 저장한 캠페인 확인
   - 상태, 생성일자 확인

3. **캠페인 불러오기**
   - "캠페인 불러오기" 버튼 클릭
   - 원하는 캠페인 선택
   - "불러오기" 클릭

## 4. 고객센터 테스트

### 4.1 공지사항 확인
1. **고객센터 접속**
   - 상단 메뉴 "고객센터" 클릭
   - 또는 http://localhost:3000/support 접속

2. **공지사항 탭**
   - 기본으로 공지사항 탭이 선택됨
   - 공지사항 목록 확인

3. **공지사항 상세보기**
   - 원하는 공지사항 제목 클릭
   - 상세 내용 팝업 확인
   - X 버튼으로 닫기

### 4.2 자주 묻는 질문 (FAQ)
1. **FAQ 탭 클릭**
   - "자주 묻는 질문" 탭 클릭

2. **카테고리 선택**
   - 상단 드롭다운에서 카테고리 선택
   - "회원가입/로그인" 선택

3. **검색 기능**
   - 검색창에 "비밀번호" 입력
   - 관련 FAQ만 필터링되는지 확인

4. **FAQ 펼치기**
   - 질문 클릭하여 답변 확인
   - 다시 클릭하여 접기

### 4.3 문의하기
1. **문의하기 탭 클릭**
   - "문의하기" 탭 클릭

2. **문의 작성**
   - 문의 유형: "서비스 이용" 선택
   - 제목: "테스트 문의입니다"
   - 내용: 상세한 문의 내용 작성
   
3. **파일 첨부 (선택)**
   - "파일 첨부" 버튼 클릭
   - PDF, JPG, PNG 파일 선택

4. **SMS 알림 설정**
   - "답변 완료 시 SMS로 알려드립니다" 체크

5. **문의 등록**
   - "문의하기" 버튼 클릭
   - 성공 메시지 확인

## 5. 마이페이지 테스트

### 5.1 회원정보 변경
1. **마이페이지 접속**
   - 상단 프로필 아이콘 클릭 → "마이페이지"
   - 또는 http://localhost:3000/my-site/advertiser/profile 접속

2. **회원정보 변경 탭**
   - 기본으로 선택되어 있음
   - 현재 정보 확인

3. **정보 수정**
   - 전화번호 변경: 010-9999-8888
   - "저장" 버튼 클릭
   - 성공 메시지 확인

### 5.2 비밀번호 변경
1. **비밀번호 변경 탭 클릭**

2. **비밀번호 입력**
   - 현재 비밀번호: Test@1234
   - 새 비밀번호: NewTest@5678
   - 비밀번호 확인: NewTest@5678

3. **변경하기**
   - "변경하기" 버튼 클릭
   - 성공 메시지 확인

### 5.3 발신번호 관리
1. **발신번호 관리 탭 클릭**

2. **발신번호 추가**
   - "발신번호 추가" 버튼 클릭
   - 발신번호: 02-1234-5678
   - 번호 유형: "대표번호"
   - "저장" 버튼 클릭

3. **기본 발신번호 설정**
   - 목록에서 "기본으로 설정" 버튼 클릭
   - 기본 표시 확인

### 5.4 세금계산서 설정
1. **세금계산서 탭 클릭**

2. **담당자 정보 입력**
   - 담당자 이메일: tax@company.com
   - 담당자명: 김담당
   - 연락처: 02-1234-5678

3. **저장**
   - "저장" 버튼 클릭
   - 성공 메시지 확인

## 🔧 문제 해결 가이드

### 로그인이 안 되는 경우
1. 아이디/비밀번호 확인
2. Caps Lock 확인
3. 브라우저 쿠키 삭제 후 재시도

### 버튼 클릭이 안 되는 경우
1. 페이지 새로고침 (F5)
2. 브라우저 캐시 삭제
3. 다른 브라우저로 시도

### 파일 업로드 실패
1. 파일 크기 확인 (10MB 이하)
2. 파일 형식 확인 (PDF, JPG, PNG)
3. 파일명에 특수문자 제거

### AI 응답이 없는 경우
1. 5-10초 기다리기
2. 페이지 새로고침 후 재시도
3. 네트워크 연결 확인
