## 📋 테스트 준비사항

### 1. 개발 서버 실행
1. 터미널을 열고 프로젝트 폴더로 이동합니다
2. 다음 명령어를 실행합니다:
```bash
cd /Users/johnlee12/Desktop/git/mts-message
npm install
npm run dev
```
3. 브라우저에서 http://localhost:3000 접속

### 2. 테스트 환경 설정
- **권장 브라우저**: Chrome 또는 Edge
- **화면 해상도**: 1920x1080 이상
- **팝업 차단 해제**: 브라우저 주소창 우측 팝업 차단 아이콘 클릭 → "항상 허용"

### 3. 테스트 계정 정보
```
일반 사용자:
아이디: testuser001
비밀번호: Test@1234

영업사원:
아이디: testsales001
비밀번호: Sales@1234

관리자:
아이디: admin001
비밀번호: Admin@1234
```

## 🧪 기능별 수동 테스트 절차

## 1. 회원가입 테스트

### 1.1 일반회원 가입
1. **회원가입 페이지 접속**
   - 브라우저에서 http://localhost:3000/signup 접속
   - 또는 로그인 페이지에서 "회원가입" 링크 클릭

2. **회원 유형 선택**
   - "일반회원" 라디오 버튼 선택
   - 추천인 코드가 있다면 입력 (선택사항)

3. **회원 정보 입력**
   - 본인인증 실행(kginicis) 시 이름, 생년월일, 휴대폰 번호 자동 입력
      - 현재 test key 사용 중이라 모두 같은 정보를 출력합니다. 
   - 이름, 생년월일, 휴대폰 번호 수정정
   - 아이디: 4~15자 이내 영문 소문자, 숫자 조합 (예: testuser002)
   - 이메일: 실제 사용하는 이메일 입력
      = 아이디, 비밀번호 찾기 시 사용
   - 비밀번호: 영문, 숫자, 특수문자 포함 8자 이상 20자 이하
      - 동일한 문자, 연속된 문자 4개 이상 사용 불가
      - 예: Test@1346
   - 비밀번호 확인: 위와 동일하게 입력

4. **추천인 정보 입력(선택)**
   - 추천인 ID

5. **약관 동의**
   - "전체 동의" 체크박스 클릭
   - 또는 개별 약관 확인 후 각각 동의

6. **회원가입 완료**
   - "회원가입" 버튼 클릭
   - 성공 메시지 확인
   - 로그인 페이지로 자동 이동 확인

### 1.2 영업사원 가입(현재 비활성 상태태)
1. 회원가입 페이지에서 "영업사원" 라디오 버튼 선택
2. 나머지 절차는 일반회원과 동일
3. 가입 후 관리자 승인 대기 상태 확인

## 2. 로그인 테스트

### 2.1 일반 로그인
1. **로그인 페이지 접속**
   - http://localhost:3000/login 접속

2. **로그인 정보 입력**
   - 아이디: user1
   - 비밀번호: test1346!

3. **로그인 시도**
   - "로그인" 버튼 클릭
   - 대시보드로 이동 확인

### 2.2 소셜 로그인
1. **카카오 로그인(앱 관리자 설정 필요)**
   - 노란색 "카카오 로그인" 버튼 클릭
   - 카카오 계정으로 로그인
   - 첫 로그인 시 추가 정보 입력

2. **네이버 로그인(앱 관리자 인증 필요)**
   - 초록색 "네이버 로그인" 버튼 클릭
   - 네이버 계정으로 로그인

3. **구글 로그인**
   - "구글 로그인" 버튼 클릭
   - 구글 계정으로 로그인

### 2.3 로그인 상태 확인
- 상단 헤더에 사용자 이름 표시 확인
- 알림 아이콘 표시 확인
- "로그아웃" 버튼 표시 확인

## 3. AI 타겟마케팅 테스트

### 3.1 초기 페이지 접속 및 퀵 채팅
1. **AI 타겟마케팅 페이지 접속**
   - 상단 메뉴에서 "AI 타겟마케팅" 클릭
   - 또는 http://localhost:3000/target-marketing 직접 접속

2. **퀵 채팅 버튼 테스트**
   - 하단에 표시되는 퀵 버튼들 확인:
     - "신메뉴 홍보", "오픈 이벤트", "단골 이벤트", "할인 이벤트", "고객유치 이벤트"
   - 각 버튼 클릭 시 해당 텍스트가 입력창에 자동 입력되는지 확인
   - "신메뉴 홍보" 버튼 클릭하여 자동 채팅 시작 확인

3. **직접 메시지 입력**
   - 입력창에 "미용실 개업 이벤트" 직접 입력
   - Enter 키로 전송 (줄바꿈은 Shift+Enter)
   - "생성" 버튼 클릭으로 전송
   - 입력창이 비워지고 로딩 상태 확인

4. **AI 응답 확인**
   - AI 응답 대기 (약 5-10초 소요)
   - 채팅 메시지가 실시간으로 스트리밍되는지 확인
   - 응답 완료 후 우측 템플릿 미리보기 영역 업데이트 확인

### 3.2 이미지 업로드 및 첨부파일 관리
1. **이미지 첨부 기능**
   - 입력창 좌측 이미지 아이콘 클릭
   - 드롭다운 메뉴에서 "이미지 업로드" 선택
   - 파일 선택 창에서 이미지 파일 선택 (JPG, PNG, 10MB 이하)

2. **첨부파일 미리보기**
   - 업로드된 이미지가 입력창 하단에 미리보기로 표시되는지 확인
   - 파일명, 크기 정보 표시 확인
   - X 버튼으로 첨부파일 삭제 가능한지 확인

3. **이미지 포함 메시지 전송(프롬프트 수정 중)**
   - 이미지가 첨부된 상태에서 메시지 전송
   - AI가 이미지를 분석하여 관련 내용을 포함한 응답 확인

### 3.3 템플릿 미리보기 및 편집
1. **템플릿 미리보기 확인**
   - AI 응답 후 우측 "템플릿 미리보기" 영역 확인
   - 생성된 문자 내용이 실제 메시지 형태로 표시되는지 확인
   - 이미지가 포함된 경우 이미지도 미리보기에 표시되는지 확인

2. **내용 직접 편집**
   - 미리보기 영역의 텍스트 클릭하여 편집 모드 진입
   - 내용 수정 후 저장
   - 수정된 내용이 반영되는지 확인

### 3.4 템플릿 저장 및 불러오기
1. **템플릿 저장**
   - 우측 템플릿 영역의 "템플릿 저장" 버튼 클릭
   - 저장 모달창 확인:
     - 템플릿 이름: "미용실 개업 홍보"
     - 카테고리 선택 드롭다운
     - 공개/비공개 설정
   - "저장" 버튼 클릭
   - 성공 메시지 확인

2. **저장된 템플릿 확인**
   - 상단 "템플릿" 탭 클릭
   - 방금 저장한 템플릿이 목록에 표시되는지 확인
   - 템플릿 카드에 제목, 내용, 저장일시 표시 확인

3. **템플릿 불러오기**
   - 저장된 템플릿 카드의 "사용하기" 버튼 클릭
   - 템플릿 내용이 미리보기 영역에 로드되는지 확인
   - 불러온 템플릿을 기반으로 추가 편집 가능한지 확인

### 3.5 캠페인 관리 및 발송 설정
1. **캠페인 불러오기**
   - 상단 "캠페인 관리" 탭 클릭
   - 저장된 캠페인 목록 확인
   - "캠페인 불러오기" 버튼 클릭
   - 원하는 캠페인 선택 후 "불러오기" 클릭
   - 캠페인 정보가 우측 설정 영역에 로드되는지 확인

3. **타겟 설정**
   - 성별, 연령대, 지역 설정(추후 AI가 추천하도록 수정 예정정)
   - 카드 이용 금액, 시간대 설정
   - 설정한 조건에 따른 예상 타겟 수 확인

### 3.6 발송 및 승인 요청
1. **발송 설정 확인**
   - 발송 일시 설정 (실시간 발송/일괄괄 발송)
   - 예상 비용 확인

2. **승인 요청**
   - 모든 설정 완료 후 "승인 요청" 버튼 클릭
   - 승인 요청 확인 모달
   - 요청 완료 후 상태 변경 확인

### 3.7 추가 기능 테스트
1. **연속 대화**
   - AI 응답 후 추가 요청하기
   - "좀 더 친근하게 수정해주세요" 입력
   - "이미지를 추가해주세요" 입력
   - 대화 히스토리가 유지되는지 확인

2. **다양한 입력 방식**
   - 긴 텍스트 입력 (1000자 이상)
   - 특수문자 포함 텍스트
   - 이모지 포함 텍스트
   - 각각의 경우에 정상 처리되는지 확인

3. **오류 상황 테스트**
   - 네트워크 연결 끊김 상황
   - AI 응답 시간 초과
   - 할당량 초과 오류 메시지 확인
   - 각 오류에 대한 적절한 안내 메시지 표시 확인

## 4. 고객센터 테스트

### 4.1 공지사항 확인
1. **고객센터 접속**
   - 상단 메뉴 "고객센터" 클릭
   - 또는 http://localhost:3000/support 접속

2. **공지사항 탭**
   - 기본으로 공지사항 탭이 선택됨
   - 공지사항 목록 확인

3. **공지사항 상세보기**
   - 원하는 공지사항 제목 클릭하여 상세 내용 확인
   - 다시 클릭하여 접기기

### 4.2 자주 묻는 질문 (FAQ)
1. **FAQ 탭 클릭**
   - "자주 묻는 질문" 탭 클릭

2. **카테고리 선택**
   - 상단 버튼 통해서서 카테고리 선택
   - "로그인" 선택

3. **검색 기능**
   - 검색창에 "비밀번호" 입력
   - 관련 FAQ만 필터링되는지 확인

4. **FAQ 펼치기**
   - 질문 클릭하여 답변 확인
   - 다시 클릭하여 접기

### 4.3 문의하기
1. **문의하기 탭 클릭**
   - "문의하기" 탭 클릭

2. **문의 작성**
   - 문의 유형: "회원정보" 선택
   - 제목: "테스트 문의입니다"
   - 내용: 상세한 문의 내용 작성
   
3. **파일 첨부 (선택)**
   - "파일 첨부" 버튼 클릭
   - PDF, JPG, PNG 파일 선택

4. **SMS 알림 설정**
   - "답변 완료 시 SMS로 알려드립니다" 체크

5. **문의 등록**
   - "문의하기" 버튼 클릭
   - 성공 메시지 확인

## 5. 마이페이지 테스트

### 5.1 회원정보 변경
1. **마이페이지 접속**
   - 상단 프로필 아이콘 클릭 → "마이페이지"
   - 또는 http://localhost:3000/my-site/advertiser/profile 접속

2. **회원정보 변경 탭**
   - 기본으로 "회원정보" 탭이 선택되어 있음
   - 현재 정보 확인 (이름, 이메일, 휴대폰 번호, 가입일)

3. **휴대폰 번호 변경**
   - "휴대폰 번호 변경" 버튼 클릭
   - 본인인증 진행 (현재는 테스트 환경으로 자동 변경)
   - 인증 완료 후 번호 업데이트 확인

4. **마케팅 정보 수신 동의 설정**
   - "SMS 마케팅 정보 수신 동의" 체크박스 토글
   - "이메일 마케팅 정보 수신 동의" 체크박스 토글
   - 변경사항 자동 저장 확인

5. **SNS 계정 연동**
   - 카카오톡 "연동하기" 버튼 클릭 → 카카오 로그인 진행
   - 네이버 "연동하기" 버튼 클릭 → 네이버 로그인 진행  
   - 구글 "연동하기" 버튼 클릭 → 구글 로그인 진행
   - 연동 완료 후 "연동 해제" 버튼으로 변경 확인

6. **SNS 계정 연동 해제**
   - 연동된 계정의 "연동 해제" 버튼 클릭
   - 확인 메시지 후 해제 완료

### 5.2 기업정보 관리
1. **기업정보 탭 클릭**
   - 상단 탭에서 "기업정보" 선택

2. **기업정보 확인 및 수정**
   - "기업정보 수정" 버튼 클릭
   - 회사명, 대표자명, 사업자등록번호 등 정보 입력
   - "저장" 버튼 클릭
   - 성공 메시지 확인

3. **사업자등록증 업로드**
   - "사업자등록증" 섹션에서 "파일 선택" 클릭
   - PDF 또는 이미지 파일 업로드
   - 업로드 완료 확인

4. **재직증명서 업로드**
   - "재직증명서 수정" 버튼 클릭
   - 파일 선택 후 업로드
   - 업로드 완료 확인

### 5.3 비밀번호 변경
1. **비밀번호 변경 탭 클릭**
   - 상단 탭에서 "비밀번호 변경" 선택

2. **비밀번호 입력**
   - 현재 비밀번호: Test@1234
   - 새 비밀번호: NewTest@5678
   - 비밀번호 확인: NewTest@5678

3. **비밀번호 변경 실행**
   - "비밀번호 변경" 버튼 클릭
   - 성공 메시지 확인
   - 모달 자동 닫힘 확인

### 5.4 발신번호 관리
1. **발신번호 관리 탭 클릭**
   - 상단 탭에서 "발신번호 관리" 선택
   - 등록된 발신번호 목록 확인

2. **발신번호 추가**
   - "발신번호 추가" 버튼 클릭
   - 전화번호: 02-1234-5678 입력
   - 발신번호명: "회사 대표번호" 입력 (선택사항)
   - "추가" 버튼 클릭
   - 목록에 추가 확인

## 🔧 문제 해결 가이드

### 로그인이 안 되는 경우
1. 아이디/비밀번호 확인
2. Caps Lock 확인
3. 브라우저 쿠키 삭제 후 재시도

### 버튼 클릭이 안 되는 경우
1. 페이지 새로고침 (F5)
2. 브라우저 캐시 삭제
3. 다른 브라우저로 시도

### 파일 업로드 실패
1. 파일 크기 확인 (10MB 이하)
2. 파일 형식 확인 (PDF, JPG, PNG)
3. 파일명에 특수문자 제거

### AI 응답이 없는 경우
1. **일반적인 해결방법**
   - 5-10초 기다리기 (정상 응답 시간)
   - 페이지 새로고침 후 재시도
   - 네트워크 연결 상태 확인

2. **할당량 초과 오류**
   - "OpenAI API 할당량을 초과했습니다" 메시지 확인
   - 관리자에게 문의하여 할당량 증설 요청
   - 잠시 후 재시도

3. **채팅이 진행되지 않는 경우**
   - Enter 키가 줄바꿈으로 작동하면 Shift+Enter 사용
   - "생성" 버튼이 비활성화된 경우 메시지 입력 확인
   - 브라우저 개발자 도구에서 콘솔 오류 확인

4. **초기 메시지 자동 전송 실패**
   - sessionStorage 데이터 확인 (F12 → Application → Session Storage)
   - 브라우저 캐시 및 세션 스토리지 삭제 후 재시도
   - 다른 브라우저에서 테스트

5. **이미지 업로드 관련 오류**
   - 파일 크기 10MB 이하 확인
   - 지원 형식 확인 (JPG, PNG)
   - 파일명에서 특수문자 제거
