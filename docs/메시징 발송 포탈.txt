# 📌 B2C/B2B 메시지 발송 웹포탈 구축 프로젝트 견적서

## ✅ 프로젝트 개요
* **프로젝트명**: 메시지 발송 웹포탈 개발
* **목표**: 캠페인 중심 메시지 발송 UI 및 관리 시스템 구축 (PG, 정산, 리워드 포함)
* **개발 기간**: 총 10주
* **개발 비용**: 20,000,000원 (부가세 별도)

---

## 🗂️ 주요 개발 범위

### 1. 기획 및 설계
* PC/MO 반응형 웹 흐름 정의
* 화면 플로우 및 와이어프레임 설계
* ERD 및 API 스펙 기획
* 요구사항 명세서 및 구현 범위 세부 정의

### 2. UI/UX 디자인
* 관리자 / 사용자 / 캠페인 관리 페이지 디자인
* 반응형 웹 기준 (PC/MO 대응)
* 레퍼런스 사이트 기반 UI/UX 최적화

### 3. 프론트엔드
* Next.js + TypeScript 기반 웹 애플리케이션 개발
* 캠페인 등록, 메시지 내용 작성, 대상자 업로드
* 회원가입 및 인증, 사용자 대시보드 구현
* 관리자 전용 기능 및 인터페이스 구현
* 반응형 디자인 적용 (모바일/데스크톱)

### 4. 백엔드
* Spring Boot + PostgreSQL + Java 기반 API 서버 개발
* 사용자 / 캠페인 / 결제 / 리워드 / 정산 로직 구현
* 메시지 발송 요청은 인터페이스 테이블(DB) 적재만 진행
* 관리자 API 개발 (승인, 검토, 통계 집계)
* RESTful API 설계 및 개발

### 5. 관리자 시스템
* 별도 관리자 웹 콘솔 개발
* 사용자 관리 및 권한 제어
* 캠페인 승인/반려 프로세스
* 발신번호 검수 및 승인 기능
* 기업서류 검수 및 승인 기능
* 통계 및 대시보드 제공

### 6. PG 및 정산 연동
* 카드 결제 연동 (PG사 연동)
* 영업사원 포인트/리워드 적립 로직 구현
* 세금계산서 발행용 자료 생성 기능

### 7. 영업사원 리워드 시스템 (다단계)
* 추천인 기반 고객 유입 추적 시스템
* 추천 링크 생성 및 트래킹 기능 개발
* 다단계 MLM(Multi-Level Marketing) 구조 구현
* 직접 추천 및 간접 추천에 따른 차등 수수료 구조 설계
* 계층별(레벨별) 수수료 비율 설정 기능
* 매출 연동 인센티브 자동 계산 로직
* 영업사원 대시보드 (직접/간접 추천 현황, 수익 현황, 조직도)
* 조직도 시각화 (본인 하위 판매 조직망 확인)
* 수수료 지급 주기 설정 및 자동 정산 기능
* 성과 기반 인센티브 프로그램 설계
* 리워드 발생 세부 내역 조회 및 통계 제공

### 8. 배포 및 인프라
* Docker 컨테이너 기반 개발 환경 구성
* 클라우드 환경 배포 (AWS 또는 클라이언트 요청 플랫폼)
* CI/CD 파이프라인 구축
* 프로덕션 환경 설정 및 보안 강화

## 🧑‍💻 기술 스택
* **프론트엔드**: Next.js, TypeScript, React
* **백엔드**: Spring Boot, Java, PostgreSQL
* **인프라**: Docker, AWS(또는 클라이언트 선호 클라우드)
* **CI/CD**: Jenkins 또는 GitHub Actions
* **기타**: RESTful API, JWT 인증

## 💬 관리 및 협업 방식
* **주단위 산출물 실시간 공유**: 매주 진행상황 및 산출물을 실시간으로 공유하여 투명한 커뮤니케이션 유지
* **소통 채널**: 이메일, 카카오톡, 구글 미트 등을 통한 신속한 커뮤니케이션
* **정기 보고**: 주 1회 진행상황 보고 및 이슈 논의
* **피드백 반영**: 클라이언트 피드백을 신속하게 개발에 반영
* **버전 관리**: Git을 통한 코드 버전 관리 및 이슈 추적

## 📎 특이사항 및 유의점
* 메시지 서버는 의뢰인 측에서 제공하며, 메시지 발송은 인터페이스 테이블(DB) 적재만 진행
* 외부 시스템 연동은 클라이언트 제공 스펙 기준으로 대응
* 디자인 리소스(로고, 브랜드 컬러 등) 클라이언트 제공 전제
* PG사는 클라이언트 사전 계약 완료되어야 연동 가능
* 레퍼런스 사이트(solapi.com, ppurio.com, findjob.co.kr) 참고하여 UI/UX 구성
* 영업사원 리워드 시스템은 다단계(MLM) 구조로 구현하여 직접 및 간접 추천에 따른 차등 리워드 제공
